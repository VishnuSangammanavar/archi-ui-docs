# Spinner

Text Fields let user enter and edit text.

## Basic Spinner

<Tabs items={['Preview', 'code']}>
  <Tabs.Tab value='Preview'>
    <div className="flex items-center justify-center gap-2">
      <Spinner />
    </div>
  </Tabs.Tab>
  <Tabs.Tab value='Code'>
    ```jsx
    import React from 'react';
    import Spinner from 'wind-ui/Spinner'

    export default function MyReactPage() {
      return (
        <Spinner />
      );
    }
    ```
  </Tabs.Tab>
</Tabs>

## Usage
```jsx
import Spinner from 'wind-ui/Spinner'
// or
import { Spinner } from 'wind-ui'
```
```jsx
<Spinner />
```

## Size
The `Spinner` comes with three variants: outlined(default), filled, standard.

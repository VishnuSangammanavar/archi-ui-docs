## Avatar Fallback
If there is an error loading the avatar image, the component falls back to an alternative in the following order:

- The provided children
- the first letter of the alt text
- a generic avatar icon - `<User />`

<Tabs items={['Preview', 'code']}>
  <Tabs.Tab value='Preview'>
    <AvatarFallback />
  </Tabs.Tab>
  <Tabs.Tab value='Code'>
    ```jsx showLineNumbers
    import React from 'react';
    import Avatar from 'archi-ui/Avatar'

    export default function MyReactPage() {
      return (
        <div className="flex items-center justify-center gap-2">
          // children is used here for fallback
          <Avatar alt='Eren Yeager' src='/broken-image.jpg' className='bg-error'>B</Avatar>
          // alt is used here for fallback
          <Avatar alt='Mikasa Ackerman' src='/broken-image.jpg' className='bg-success' />
          // a generic avatar icon is used here for fallback
          <Avatar src='broken-image.jpg' className='bg-secondary' />
        </div>
      );
    }
    ```
  </Tabs.Tab>
</Tabs>

## Custom surplus
Set the `renderSurplus` prop as a callback to customize the surplus avatar. The callback will receive the surplus number as an argument based on the children and the `max` prop, and should return a `React.ReactNode`.

The `renderSurplus` prop is useful when you need to render the surplus based on the data sent from the server.

<Tabs items={['Preview', 'code']}>
  <Tabs.Tab value='Preview'>
    <AvatarRenderSurplus />
  </Tabs.Tab>
  <Tabs.Tab value='Code'>
    ```jsx showLineNumbers /renderSurplus/
    "use client"

    import React from 'react';
    import Avatar from 'archi-ui/Avatar'
    import AvatarGroup from 'archi-ui/AvatarGroup'

    export default function MyReactPage() {
      return (
        <div className="flex items-center justify-center">
          <AvatarGroup total={5500} renderSurplus={(surplus) => <span>+{surplus.toString()[0]}K</span>}>
            <Avatar alt='Naruto' src='/naruto-uzumaki.webp' />
            <Avatar alt='Eren Yeager' src='/eren-jaeger.webp' />
            <Avatar alt='Mikasa Ackerman' src='/mikasa-ackermen.jpg' />
            <Avatar alt='Armin' src='/armin.jpg' />
            <Avatar alt='Levi Ackerman' src='/levi-ackermen.jpg' />
          </AvatarGroup>
        </div>
      );
    }
    ```
  </Tabs.Tab>
</Tabs>

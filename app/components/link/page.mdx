# Link

Links are used to provide accessible navigation.

## Usage
```jsx
import Link from 'archi-ui/Link'
// or
import { Link } from 'archi-ui'
```
```jsx
<Link href='#'>Link</Link>
```

## Underline
The `underline` prop can be used to set underline behavior. The default is `always`.

<Tabs items={['Preview', 'code']}>
  <Tabs.Tab value='Preview'>
    <UnderlineLink />
  </Tabs.Tab>
  <Tabs.Tab value='Code'>
    ```jsx showLineNumbers
    import React from 'react';
    import Link from 'archi-ui/Link'

    export default function MyReactPage() {
      return (
        <div className='flex items-center justify-center gap-2'>
          <Link href='#' underline='none'>Underline-none</Link>
          <Link href='#' underline='hover'>Underline on hover</Link>
          <Link href='#' underline='always'>Underline always</Link>
        </div>
      );
    }
    ```
  </Tabs.Tab>
</Tabs>

## Color

# Divider

The divider component provides a thin, unobtrusive line for grouping elements to reinforce visual hierarchy.

# Introduction
The Archi-UI Divider component renders as a dark gray `<hr>` by default, and features several useful props for quick style adjustments.

## Variant
The Divider component supports three variants: `fullwidth`(default), `inset`, and `middle`.

            </ListItem>

## Orientation
Use the `orientation` prop to change the Divider from horizontal to vertical. When using vertical orientation, the Divider renders a `<div>` with the corresponding accessibility attributes instead of `<hr>`.

<Tabs items={['Preview', 'code']}>
  <Tabs.Tab value='Preview'>
    <DividerOrientation />
  </Tabs.Tab>
  <Tabs.Tab value='Code'>
    ```jsx showLineNumbers
    import React from 'react';
    import Card from 'archi-ui/Card'
    import Divider from 'archi-ui/Divider'
    import Typography from 'archi-ui/Typography'
    import { AlignCenter, AlignLeft, AlignRight, Bold, Italic } from 'lucide-react'

    export default function MyReactPage() {
      return (
        <div className='flex flex-col items-center justify-center gap-4'>
          <Card variant='outlined' className='flex items-center justify-center gap-2 px-2 py-1'>
            <Typography>Cut</Typography>
            <Divider orientation='vertical' className='h-5' />
            <Typography>Copy</Typography>
            <Divider orientation='vertical' className='h-5' />
            <Typography>Paste</Typography>
            <Divider orientation='vertical' className='h-5' />
            <Typography>Select All</Typography>
          </Card>
          <Card variant='outlined' className='flex items-center justify-center gap-2 px-2 py-1'>
            <AlignLeft />
            <AlignCenter />
            <AlignRight />
            <Divider orientation='vertical' className='h-5' />
            <Bold />
          </Card>
          <Card variant='outlined' className='flex items-center justify-center gap-2 px-2 py-1'>
            <Bold />
            <Divider orientation='vertical' className='h-5' />
            <Italic />
          </Card>
        </div>
      );
    }
    ```
  </Tabs.Tab>
</Tabs>

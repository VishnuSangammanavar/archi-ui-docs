# Checkbox

Checkboxs allow the user to select one or more items from a set.

If you have multiple options appearing in a list, you can preserve space by using checkboxes instead of on/off switches. If you have a single option, avoid using a checkbox and use an on/off switch instead.

## Indeterminate
A checkbox input can only have two states in a form: checked or unchecked. It either submits its value or doesn't. Visually, there are three states a checkbox can be in: checked, unchecked, or indeterminate.

You can change the indeterminate icon using the `indeterminateIcon` prop.

<Tabs items={['Preview', 'code']}>
  <Tabs.Tab value='Preview'>
    <IndeterminateIcon />
  </Tabs.Tab>
  <Tabs.Tab value='Code'>
    ```jsx showLineNumbers
    "use client"

    import React, { useState } from 'react'
    import { Checkbox, FormControl, Label } from 'archi-ui'

    export default function MyReactPage() {

      const [checked, setChecked] = useState([true, false]);

      const handleChange1 = (event) => {
        setChecked([event.target.checked, event.target.checked]);
      };

      const handleChange2 = (event) => {
        setChecked([event.target.checked, checked[1]]);
      };

      const handleChange3 = (event) => {
        setChecked([checked[0], event.target.checked]);
      };
      return (
        <div className='flex flex-col items-center justify-center gap-2'>
          <FormControl>
            <Checkbox
              checked={checked[0] && checked[1]}
              indeterminate={checked[0] !== checked[1]}
              onChange={handleChange1}
            />
            <Label htmlFor='parent'>Parent</Label>
          </FormControl>
          <div className="flex flex-col ml-4 gap-2">
            <FormControl>
              <Checkbox checked={checked[0]} onChange={handleChange2} />
              <Label htmlFor='child1'>Child1</Label>
            </FormControl>
            <FormControl>
              <Checkbox checked={checked[1]} onChange={handleChange3} />
              <Label htmlFor='child2'>Child2</Label>
            </FormControl>
          </div>
        </div>
      );
    }
    ```
  </Tabs.Tab>
</Tabs>

## Controlled
You can control the checkbox with the `checked` and `onChange` props.

<Tabs items={['Preview', 'code']}>
  <Tabs.Tab value='Preview'>
    <Controlled />
  </Tabs.Tab>
  <Tabs.Tab value='Code'>
    ```jsx showLineNumbers
    "use client"

    import React, { useState } from 'react';
    import { Checkbox } from 'archi-ui'

    export default function MyReactPage() {

      const [check, setCheck] = useState(false);

      const handleCheck = () => {
        setCheck(!check)
      }

      return (
        <div className="flex items-center justify-center gap-2">
          <Checkbox checked={check} onChange={handleCheck} />
          <p>Checkbox is checked: {String(check)}</p>
        </div>
      );
    }
    ```
  </Tabs.Tab>

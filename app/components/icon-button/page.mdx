# IconButton

Icon buttons are commonly found in app bars and toolbars

Icons are also appropriate for toggle buttons that allow a single choice to be selected or deselected, such as adding or removing a star to an item.

## Basic button
<Tabs items={['Preview', 'code']}>
  <Tabs.Tab value='Preview'>
    <BasicIconButton />
  </Tabs.Tab>
  <Tabs.Tab value='Code'>
    ```jsx showLineNumbers

    import IconButton from 'wind-ui/IconButton'
    import { AlarmClock, ShoppingCart, Trash } from 'lucide-react'

      return (
        <div className='flex items-center justify-center'>
    ```

## Usage
```jsx
import IconButton from 'wind-ui/IconButton'
import { Heart } from 'lucide-react'
// or
import { IconButton } from 'wind-ui'
import { Heart } from 'lucide-react'
```
```jsx
<IconButton color='error'>
  <Heart />
</IconButton>
```

## Hyperlink IconButton
To use the `IconButton` component as hyperlink, just pass `href` prop or change the root element to `<a>` tag using `component` prop.

If `href` prop is passed, an `<a>` element will be used as the root node.

<Tabs items={['Preview', 'code']}>
  <Tabs.Tab value='Preview'>
    <HyperlinkIconButton />
  </Tabs.Tab>
  <Tabs.Tab value='Code'>
    ```jsx showLineNumbers
    import React from 'react';
    import IconButton from 'wind-ui/IconButton'
    import { Facebook } from 'lucide-react'

    export default function MyReactPage() {
      return (
        <div className='flex items-center justify-center'>
          <IconButton component='a' href='#' color='primary'><Facebook /></IconButton>
        </div>
      );
    }
    ```
  </Tabs.Tab>
</Tabs>

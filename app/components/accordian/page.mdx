# Accordion

The Accordion component lets users show and hide sections of related content on a page.

- Accordion: The wrapper for grouping related components.
- Accordion Summary: The wrapper for the Accordion header, which expands or collapses the content when clicked.
- Accordion Details: The wrapper for the Accordion content.
- Accordion Actions: an optional wrapper that groups a set of buttons.

## Controlled Accordion
The Accordion component can be controlled or uncontrolled.

<Tabs items={['Preview', 'code']}>
  <Tabs.Tab value='Preview'>
    <ControlledAccordion />
  </Tabs.Tab>
  <Tabs.Tab value='Code'>
    ```jsx showLineNumbers
    import React from 'react';
    import Accordion from 'elementx/Accordion'
    import AccordionSummary from 'elementx/AccordionSummary'
    import AccordionDetails from 'elementx/AccordionDetails'
    import Typography from 'elementx/Typography'

    export default function MyReactPage() {

      const [expanded, setExpanded] = useState(false);

      const handleChange = (panel) => (event, isExpanded) => {
        setExpanded(isExpanded ? panel : false);
      }

      return (
        <div className="flex flex-col items-center justify-center gap-2">
          <Accordion expanded={expanded === 'panel1'} onChange={handleChange('panel1')}>
            <AccordionSummary>
              <Typography component='span'>Accordion 1</Typography>
            </AccordionSummary>
            <AccordionDetails>Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolor animi harum error amet porro fugiat obcaecati molestias similique veniam aliquam commodi, quae et atque voluptas exercitationem repellat soluta? Repudiandae, repellendus?</AccordionDetails>
          </Accordion>
          <Accordion  expanded={expanded === 'panel2'} onChange={handleChange('panel2')}>
            <AccordionSummary>
              <Typography component='span'>Accordion 2</Typography>
            </AccordionSummary>
            <AccordionDetails>Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolor animi harum error amet porro fugiat obcaecati molestias similique veniam aliquam commodi, quae et atque voluptas exercitationem repellat soluta? Repudiandae, repellendus?</AccordionDetails>
          </Accordion>
          <Accordion expanded={expanded === 'panel3'} onChange={handleChange('panel3')}>
            <AccordionSummary>
              <Typography component='span'>Accordion with actions</Typography>
            </AccordionSummary>
            <AccordionDetails>Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolor animi harum error amet porro fugiat obcaecati molestias similique veniam aliquam commodi, quae et atque voluptas exercitationem repellat soluta? Repudiandae, repellendus?</AccordionDetails>
          </Accordion>
        </div>
      );
    }
    ```
  </Tabs.Tab>

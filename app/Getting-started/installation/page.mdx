# Installation

## Building Beautiful and Accessible UIs with Ease

How to install and set up Archi-UI in your project.

## Installing through NPM
Archi-UI is distributed as an NPM package. You can add it to your project using your preferred package manager.

<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tabs.Tab value='npm'>
    ```bash
    npm install archi-ui
    ```
  </Tabs.Tab>
  <Tabs.Tab value='yarn'>
    ```bash
    yarn add archi-ui
    ```
  </Tabs.Tab>
  <Tabs.Tab value='pnpm'>
    ```bash
    pnpm add archi-ui
    ```
  </Tabs.Tab>
</Tabs>


## Peer dpendencies
Please note that react and react-dom are peer dependencies, meaning you should ensure they are installed before installing Archi-UI.

```bash showLineNumbers
"peerDependencies": {
  "react": "^17.0.0 || ^18.0.0 || ^19.0.0",
  "react-dom": "^17.0.0 || ^18.0.0 || ^19.0.0"
},
```

## Set Up [Tailwind CSS](https://www.tailwindcss.com/)
Archi-UI uses Tailwind CSS for styling. Make sure you have it configured in your project. If not, install and configure it using the steps below, or Follow the [Tailwind CSS](https://www.tailwindcss.com/) installation instructions to get started:

```bash
npm install -D tailwindcss @tailwindcss/postcss postcss
```

### Configure PostCSS Plugins
create a `postcss.config.mjs` file in root of your project and add the `@tailwindcss/postcss` plugin to your PostCSS configuration.

```bash showLineNumbers
const config = {
  plugins: {
    "@tailwindcss/postcss": {},
  },
};

export default config;
```

### Import Tailwind CSS
Add an '@import' to 'styles/globals.css' that imports Tailwind CSS.

```bash
@import "tailwindcss";
```
